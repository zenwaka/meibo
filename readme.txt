======================================
若手議会議員の会 議員名簿
======================================

本プロジェクトは、Googleスプレッドシート（CSV形式）のデータをもとに、名簿を表示するWebサイトです。
Netlifyにデプロイして公開することを前提としており、独自デザインのパスワード認証（Edge Functions）を備えています。

■ サイトのパスワード認証について
このサイトにはアクセス制限がかかっており、閲覧するためにはログイン画面で「合言葉（パスワード）」を入力する必要があります。
ソースコードの安全性を保つため、パスワードはソースコード（GitHub）上には一切保存せず、Netlifyの「環境変数（Environment variables）」の仕組みを利用して管理されています。


■ パスワードの設定・変更方法（Netlify環境変数）
パスワードを変更したい場合や、新しくサイトを立ち上げ直す場合は、以下の手順で Netlify の管理画面から設定を行なってください。ソースコードを直接編集する必要はありません。

1. Netlifyの管理画面（ダッシュボード）へログインします。
2. 対象のサイト（プロジェクト）をクリックします。
3. 左側のメニューから 「Site configuration」 をクリックし、その中の 「Environment variables」 を開きます。
4. 「Add a variable」（新しく追加する場合）または、既存の変数の「Edit」をクリックします。
5. 以下の通りに Key（キー）と Value（値）を設定します。
   ---------------------------------
   Key   : SITE_PASSWORD
   Value : 設定したいパスワード （現在の初期値: Q7@fL2!xP9zK など）
   ---------------------------------
6. 入力が完了したら、「Save」をクリックして保存します。
7. 【重要】 環境変数の変更を実際のサイトに反映させるため、手動で再デプロイを行ってください。
   上部メニューの「Deploys」を開き、「Trigger deploy」→「Deploy site」等をクリックすると、数十秒程度で新しいパスワードが有効になります。

※ 補足事項 ※
もし誤って「SITE_PASSWORD」の環境変数を削除してしまったり未設定の状態になった場合、安全装置（フェールセーフ）が働き、誰からのアクセスであっても「設定エラー」画面となり内部データが保護される仕組みになっています。

■ 名簿データの追加と更新（Googleスプレッドシート連携）について
このサイトは名簿データを内部には持たず、Googleスプレッドシートから直接「CSV形式」で最新データをフェッチ（取得）して表示する仕組みになっています。
データの追加や内容の更新を行うには、主に以下の2つの方法があります。

【方法1：Googleフォームからの新規登録】
新しく名簿情報を追加する場合は、以下の専用フォームから入力・送信してください。送信された内容は自動的に連携しているスプレッドシートへ登録されます。
登録フォームURL: https://docs.google.com/forms/d/e/1FAIpQLSfOzht-fTbIEGAd0prqOMjAL6jOW-OjQOM9_1dfY0-UjD768A/viewform?usp=dialog

【方法2：スプレッドシートの直接編集（情報の更新や削除）】
既存のデータの修正、掲載内容の更新、または不要なデータの削除を行いたい場合は、対象のGoogleスプレッドシートを直接開いて内容を書き換えてください。

※ どちらの方法で更新を行なった場合でも、スプレッドシート側の情報が更新されれば自動的にサイトへ反映されるため、サイトの再デプロイ（データの上げ直し）などは必要ありません。

【データ参照元のスプレッドシートを変更する手順】
もし、読み込み元のスプレッドシートそのものを新しく作り直したり別のものに変更したい場合は、以下の手順で設定内のURLを書き換えてください。

1. 新しいGoogleスプレッドシートを開き、上部メニューの「ファイル」＞「共有」＞「ウェブに公開」を選択します。
2. ドロップダウンから対象のシートを選択し、形式を「ウェブページ」から「カンマ区切り（.csv）」に変更して「公開」をクリックします。
3. 発行されたURL（ https://docs.google.com/spreadsheets/.../pub?output=csv のような形式）をすべてコピーします。
4. このプロジェクト内にある `index.html` ファイルをエディタで開きます。
5. ファイル中盤付近（<script>タグの中）にある `CSV_URL` という項目の値を、コピーした新しいURLへ書き換えます。
   ※変更例：
   const CSV_URL = "ここに新しいスプレッドシートのCSV公開URLを貼り付けます";
6. 変更を上書き保存し、GitHubにプッシュ（アップロード）します。
7. 以降はNetlifyで自動的に最新版がデプロイされ、新しいスプレッドシートのデータが読み込まれるようになります。

※ 注意事項 ※
スプレッドシートの1行目（ヘッダー行）は、「名前」「ふりがな」「都道府県」など、サイトのシステムが列を判断するための重要なキーとして使われます。
この行の名前を変更したり、不要と思って削除してしまうと、サイトの表にデータが正しく表示されなくなる等のエラーに繋がるためご注意ください。
